# AI-Маркетолог Design System: Obsidian & Lime Glassmorphism

## Colors
- Base shell: #000000
- Surface: #0c0c0c
- Primary accent: #ccff00 (Neon Lime)
- Secondary accent: #10b981 (Emerald)
- Text primary: #ebebeb
- Text secondary: rgba(255,255,255,0.6)
- Text muted: rgba(255,255,255,0.3)
- Glass bg: rgba(255,255,255,0.03)
- Glass border: rgba(255,255,255,0.1)
- Glass hover border: rgba(204,255,0,0.4)

## Typography
- Headings: Space Grotesk (300-700), letter-spacing: -0.06em
- Body: Space Grotesk (400)
- Technical labels: JetBrains Mono, uppercase, letter-spacing: 0.2em, 10px
- Hero heading: 7.5rem desktop / 3rem mobile, line-height: 0.85

## Glassmorphism
- background: rgba(255, 255, 255, 0.03)
- backdrop-filter: blur(16px)
- border: 1px solid rgba(255, 255, 255, 0.1)
- border-radius: 1.5rem cards, 2.5rem main shell
- box-shadow: inset 0 1px 0 rgba(255,255,255,0.06)

## Layout
- Floating shell: max-width 1600px, rounded 2.5rem, ring-1 white/10
- Bento grid for features
- 60x60px grid pattern background
- 15% noise SVG overlay

## Buttons
- Primary: bg #ccff00, color #000, rounded-full, shadow 0 0 30px rgba(204,255,0,0.3)
- Hover: scale 1.05
- Secondary: glass with white/10 border

## Decorative
- Glow spheres: radial-gradient, 120px blur filter
- Float animation: 6s infinite ease-in-out
- Pulsing lime dot for status indicators
- Noise texture overlay
